Intrucciones:

-Al ejecutar el programa, lo primero a hacer es crear una base de datos
	al abrir el formulario, este le solicitara, nombre, usuario y contraseña
	la base de datos puedes tener cualquier nombre
	el usuario y la contraseña deben ser correspondientes a un usuario SQL ya 	
	creado y con todos los permisos necesarios en el equipo.

-Si se hace correctamente, deberia de crear una base de datos con el nombre que usted eligio,
	y cargar las tablas y datos del archivo "database.sql" que se encuentra dentro del proyecto
	ademas de guardar los datos de la base de datos en 3 archivos .txt tambien guardados en la carpeta del proyecto
	si estuvieran vacios no deberia de permitir hacer alguna otra accion que no sea crear la base de datos

-Si todo ha salido bien hasta ahora, podra usar el programa con normalidad, debera primero iniciar sesion, ya que si no lo hace no tendra acceso a nada mas,
	si las tablas se cargaron correctamente, habra 3 usuarios ya creados:

	+De tipo Admin:
		Correo: admin@gmail.com
		contraseña: admin

	+De tipo Empleado:
		Correo: empleado@gmail.com
		contraseña: empleado

	+De tipo Cliente:
		Correo: cliente@gmail.com
		contraseña: cliente

	dependiendo de con cual use tendra acceso o no, a las demas partes del programa, puede usar "Crear Usuario" para crear otros,
	cerrar su sesion actual e iniciar sesion con el usuario que creó y usarlo dependiendo del nivel de acceso que le haya otorgado.

-Si todo funciona como deberia hasta ahora, y cierra el programa, no deberia de tener que crear otra base de datos, a menos que la haya borrado,
	la informacion deberia haberse guardado en sus respectivos archivos y solo tendria que iniciar sesion otra vez. 
	Pero si lo desea puede crear otra base de datos distinta sin borrar la anterior, el programa la creara y se conectara a ella, pasandole los datos
	y dejando la anterior tal y como la dejo.
